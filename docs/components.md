# Components

This document includes specifications for commonly used components in this website.

## Hero Body Copy

Hero body copy is larger body text used in hero and feature sections. Use this when body copy needs to stand out from standard paragraph text.

- Font: Open Sans regular
- Font size: 20px (1.25rem)
- CSS class: `.hero-body-copy`

## Button

The button component comes in a few varieties: primary, secondary.
It also comes in a few sizes, small and large.
Finally, it comes in two flavors: full-width and standard.
Whatever the variety and size, the button has some consistent properties, which are as follows:

All buttons:
Font: Open Sans bold
Font alignment: centered
Vertical alignment: centered
Corner radius: 6px

Primary button:
Background color: dark gray
Stroke: #ffffff, 40% opacity, 1px weight
Font color: yellow (#FFD00F)

Secondary button:
Background color: transparent
Font color: dark gray
stroke: dark gray, 100% opacity, 1px weight

Highlight button:
Background color: yellow (#FFD00F)
Font color: dark gray
stroke: #ffffff, 100% opacity, 1px weight

Large button
Font size: 20px
Padding-y: 12px

Small button
Font size: 16px
Padding-y: 6px
